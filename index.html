Programação Web A Internet e a Web

Na Unidade Curricular de Programação Web abordaremos os principios básicos associados ao desenvolvimento para a web, tanto do lado back-end como front-end.

Aprender-se-ão as seguintes tecnologias 🌐:

  • HTML
  • CSS
  • Python
  • Django
  • JavaScript

De forma a consolidar todos os conhecimentos, será feito um laboratório semanal, de entrega obrigatória, de aplicação da matéria leccionada. Irá ser desenvolvido também um projeto.
